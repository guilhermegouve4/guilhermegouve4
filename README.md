## 💼 Experiência Profissional

### 🤖 Sistemas Backend com Inteligência Artificial

Desenvolvimento de plataformas enterprise para assistentes inteligentes e análise financeira automatizada:

<br>

<details open>
<summary><b>🔹 Sistema de Gerenciamento de Assistentes Inteligentes</b></summary>

<br>

**Contexto:** Plataforma de copilots empresariais necessitava de interface administrativa para gestão de assistentes.

**Solução Implementada:**
- ✅ API RESTful completa para CRUD de assistants com validação robusta via DTOs
- ✅ Sistema de autorização multinível com ASP.NET Core Identity (`[Authorize]`)
- ✅ Relacionamento owner-assistant com subcoleções MongoDB otimizadas
- ✅ Gerenciamento de modelos IA e avatars com persistência eficiente
- ✅ Cobertura de testes unitários e de integração para todos os endpoints

**Tecnologias:** C#, .NET 9, MongoDB, xUnit, JWT Authentication

<br>

</details>

<details>
<summary><b>🔹 Plataforma de Diagnóstico Financeiro SmartFIDC</b></summary>

<br>

**Contexto:** Sistema de análise financeira perdia dados de modificações entre sessões de usuário.

**Solução Implementada:**
- ✅ Arquitetura de versionamento histórico com zero perda de dados
- ✅ Sistema de eventos para notificações backend em tempo real
- ✅ Subcoleções MongoDB para auditoria completa de alterações
- ✅ Persistência de estado garantindo consistência entre sessões
- ✅ Pipeline de agregação otimizado para consultas históricas

**Tecnologias:** C#, ASP.NET Core, MongoDB, Event-Driven Architecture

<br>

</details>
